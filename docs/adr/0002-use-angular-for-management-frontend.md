# ADR-0002: Use Angular For The Management Frontend

## Status

Accepted

## Context

The platform needs a management frontend for authenticated authors and administrators to manage folders, content instances, content types, file metadata, and publication workflows.

## Decision

ECMP will use Angular and TypeScript for the Management Frontend.

## Consequences

Angular provides a structured framework suitable for a complex management UI, route guards, forms, testing, and long-lived enterprise-style applications. The project will follow Angular conventions and use Angular Router, Angular Forms, and Angular HTTP Client.
